# Strong Standard — Website

**Domain:** strongstandard.com  
**Hosted:** GitHub Pages

## Structure

- `index.html` — Homepage
- `case-studies/` — Client case study pages (×11)
- `freebies/` — Lead magnet funnels (Blueprint, Meal Plan, Hyper Training)
- `schedule/` — Schedule call page (VSL + calendar)
- `confirmation/` — Call confirmation page
- `21-day-challenge/` — 21DC funnel pages
- `group-coaching/` — Group coaching checkout pages
- `legal/` — Terms, Privacy Policy
- `assets/` — Shared CSS, JS, images

## Adding a page

1. Drop your `.html` file in the correct folder
2. Add `<link rel="stylesheet" href="/assets/css/brand.css">` in the `<head>`
3. Add `<script src="/assets/js/main.js"></script>` before `</body>`
